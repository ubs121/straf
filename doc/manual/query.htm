Асуулга.

Асуулга гэдэг нь øгøгдлийн сангаас øгøгдлийг гарган авах командуудын цуглуулга юм. Програм дээр асуулгын цонхыг гаргахдаа Харах цэсний Асуулга (эсвэл гарнаас Ctrl + Shift + Q товчлуурын хослолыг ашиглаж болно) командыг сонгоно. Асуулгын командаа оруулаад Асуулга цэсний Харах командыг сонгож асуулгын vр дvнг харна.Асуулгын vр дvн нь хvснэгт хэлбэрээр харагдана. Тайлан гаргахад ашиглагдах асуулгын vндсэн команд нь Select юм.
Select командыг дараах байдлаар хэрэглэнэ.
Select талбар1, талбар2 ... from хvснэгт1, хvснэгт2 ... [where нøхцøл] [group by талбар1, талбар2, ...] [order by талбар1, талбар2, ...[asc|| desc]]

Group by øгøгдлvvдийг бvлэглэх
Order by øгøгдлvvдийг эрэмбэлэх
